# 2021D 摄像机入侵检测 — 运行说明

基于背景建模的运动检测 + 本地/HTTP 告警。配置见 **config.yaml**，联网联调见 **README.md**。

---

## 一、环境与依赖

```powershell
cd 本目录
pip install -r requirements.txt
```

---

## 二、实机摄像头

```powershell
python main.py --real --source 0
```

或（不显式写 `--real` 时）：

```powershell
python main.py --source 0
```

使用视频文件代替摄像头：

```powershell
python main.py --source video.mp4
```

---

## 三、保存检测结果视频（可选）

```powershell
python main.py --real --source 0 --save-video out.mp4
```

---

## 四、联网告警联调

1. 终端 A：`python alert_receiver.py`
2. 编辑 **config.yaml**：`network.enabled: true`，`network.url` 指向接收端（如 `http://127.0.0.1:8080/alert`）
3. 终端 B：`python main.py --real --source 0`

---

## 五、硬件清单（参考）

- 香橙派/树莓派 + USB 摄像头（固定安装）
- 网络（可选，用于 HTTP 上报）
- 路由器/交换机按现场搭建

---

## 六、常见问题

| 现象 | 处理 |
|------|------|
| 误报多 | 调 `config.yaml` 中 `threshold`、`min_area`、ROI |
| 上报失败 | 检查 URL、防火墙、requests 超时 |
| 无窗口运行 | `--no-display`（适合无屏服务器） |
