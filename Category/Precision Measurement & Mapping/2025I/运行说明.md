# 2025I 非接触式控制盘 — 运行说明

手势位置 + 握拳/张开，映射为归一化控制量。详见 **README.md**。

---

## 一、环境与依赖

- Python 3.8+
- `pip install -r requirements.txt`（含 **pyserial**，用于串口输出）

---

## 二、纯软件模拟

```powershell
cd 本目录
pip install -r requirements.txt

python main.py --simulate
python main.py --video 你的视频.mp4
```

---

## 三、实机摄像头

```powershell
python main.py --cam 0 --width 640 --height 480
```

调肤色与手势阈值：编辑 **config.py**（`SKIN_*`、`FIST_EXTENT_THRESH` 等）。调试时可加 `--show-mask` 看肤色二值图。

---

## 四、串口输出（接 MCU）

每行输出：`nx,ny,state`（state 为 `open` / `fist` / `none` 等）。

```powershell
python main.py --cam 0 --serial COM3 --baud 115200
```

Linux 串口示例：`--serial /dev/ttyUSB0`。

---

## 五、硬件清单（参考）

- 香橙派/树莓派 + USB 摄像头
- 可选：USB 转串口模块、下位机 MCU

---

## 六、常见问题

| 现象 | 处理 |
|------|------|
| 串口打不开 | 确认端口号与波特率；Linux 用户权限、`dialout` 组 |
| 手势乱跳 | 增大 `MAP_DEADZONE`、增大 `MAP_SMOOTH`；改善光照 |
| 肤色分割失败 | 微调 `SKIN_Cr/Cb` 范围；避免强背光 |
