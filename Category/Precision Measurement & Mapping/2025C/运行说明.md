# 2025C 单目视觉目标测量 — 运行说明

基于 **Orange Pi + OpenCV** 的单目测量流程。原理与目录结构见 **README.md**。

---

## 一、环境与依赖

- Python 3.7+
- `pip install -r requirements.txt`

---

## 二、纯仿真（无相机）

```powershell
cd 本目录
pip install -r requirements.txt

python main.py
```

会生成仿真图并输出 `output_sim.png`（保存目录可用 `--save-dir`）。

---

## 三、单张图片测量

```powershell
python main.py path/to/image.jpg --save-dir .
```

---

## 四、实机摄像头实时测量

1. **建议先做棋盘格标定**（提高精度）：

   ```powershell
   python run_calibration.py calibration/图1.jpg calibration/图2.jpg ...
   ```

   生成 `calibration/camera_params.npz` 后，程序会自动加载。

2. **实时运行**（按 **Q** 退出）：

   ```powershell
   python main.py --real --cam 0 --width 640 --height 480 --plane-distance 0.5
   ```

- `--plane-distance`：目标所在平面与相机距离（米），需与现场一致。
- `--max-frames`：仅测若干帧便于调试。

---

## 五、硬件与配置要点

- **硬件**：香橙派 + USB 摄像头 + 支架；标定棋盘格；可选已知尺寸参照物。
- **配置**：**config.py** 中 `DEFAULT_CAMERA_HEIGHT_M`、`REFERENCE_OBJECT_REAL_HEIGHT_M`、圆/矩形检测阈值等。

---

## 六、常见问题

| 现象 | 处理 |
|------|------|
| 测距偏差大 | 先完成标定；核对 `--plane-distance`；用已知尺寸物体验证 |
| 圆/矩形检不出 | 调 `CIRCLE_PARAM1/2`、`MIN_RADIUS/MAX_RADIUS`、边缘参数 |
| 摄像头打不开 | 换 `--cam`；Windows 用设备管理器确认编号 |
