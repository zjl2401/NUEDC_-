# 2017C 运行说明

## 左侧文件树是灰的 / 看不到文件？

1. **重新打开文件夹**  
   菜单 **文件 → 打开文件夹**，选到 **2017C** 这一层（即能看到 `main.py`、`config.py` 的文件夹），不要只选到上一级。

2. **或用工作区打开**  
   - 若在 2017C 里：双击打开 **2017C.code-workspace**；  
   - 若在仓库根目录 NUEDC 里：双击打开 **NUEDC.code-workspace**，左侧会出现「2017C-四旋翼探测跟踪」一项，点开即可。

3. **确认打开的是 2017C**  
   左侧顶部应显示「2017C」或「2017C-四旋翼探测跟踪」，下面能看到 `main.py`、`vision`、`control`、`flight` 等，才说明当前是 2017C 项目。

## 一、确认文件齐全

在 **2017C** 文件夹内应有：

- `main.py`、`config.py`、`simulate.py`、`camera.py`
- `requirements.txt`、`run.bat`
- 文件夹：`vision`、`control`、`flight`

## 二、运行方式

### 方式 A：双击运行

1. 进入 **2017C** 文件夹  
2. 双击 **run.bat**  
3. 出现窗口后为纯软件模拟，按 **Q** 退出  

### 方式 B：命令行

```powershell
# 进入 2017C（请改成你电脑上的实际路径）
cd "C:\Users\36255\Desktop\GIT\NUEDC\Category\Autonomous Flight & Obstacle Avoidance\2017C"

# 首次运行安装依赖
pip install -r requirements.txt

# 纯软件模拟
python main.py --mode simulate
```

按 **Q** 退出。

## 三、实机运行（MAVLink 飞控）

**硬件**：香橙派 + 摄像头 + 支持 MAVLink 的四旋翼飞控（数传串口或 UDP）。详见 **README.md** 硬件清单。

**依赖**：

```powershell
pip install -r requirements.txt
```

**配置**：编辑 `config.py` 中的 `FLIGHT_SERIAL_PORT` / `FLIGHT_BAUD`，或设置 `FLIGHT_CONNECTION_STRING`（如 `udp:127.0.0.1:14550`）。首次务必减小 `FLIGHT_VEL_MAX_MS`。

**命令示例**：

```powershell
# 仅摄像头视觉调试（不发送飞控速度）
python main.py --mode vision --vision-mode track --camera-index 0

# 完整任务 + 实机（Windows 串口示例）
python main.py --mode full --real --camera-index 0 --connection COM3 --baud 57600

# 飞控已手动起飞进入 GUIDED，只跑跟踪与速度
python main.py --mode full --real --skip-flight --camera-index 0 --connection COM3
```

**安全**：必须在空旷场地、螺旋桨加防护；地面站保持可急停；PX4 与 ArduPilot 模式差异大，实机以 ArduPilot Copter + GUIDED 为主。

## 四、若文件夹仍为空

说明 Cursor 里的 2017C 和你本机打开的 2017C 可能不是同一路径：

1. 在 Cursor 左侧找到 **2017C** 项目，看顶部地址栏或右键 2017C → “Reveal in File Explorer”，确认实际路径。  
2. 若 2017C 在别处（例如另一个盘或另一个 NUEDC 副本），把 **整个 2017C 文件夹**（含 vision、control、flight）从 Cursor 对应的那个路径**复制**到你当前用的 NUEDC 下的 `3UAV Navigation Firefighting in Complex Environments\2017C`。  
3. 或在 Cursor 里用 **文件 → 打开文件夹** 打开你本机用的那个 NUEDC 根目录，再让我把 2017C 的文件生成到那个路径下。

## 五、常见报错

| 报错 | 处理 |
|------|------|
| `No module named 'config'` | 必须在 2017C 目录下运行，或双击 run.bat。 |
| `No module named 'cv2'` | 执行 `pip install opencv-python numpy`。 |
| `找不到路径` | 把命令里的路径改成你电脑上 2017C 的真实路径。 |
| `实机模式需要安装 pymavlink` | 执行 `pip install pymavlink`。 |
| 串口打不开 | 检查 `--connection` 与波特率；Linux 需用户权限或 `dialout` 组。 |
