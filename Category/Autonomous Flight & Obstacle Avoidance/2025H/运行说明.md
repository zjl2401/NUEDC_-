# 2025H 野生动物巡查 — 运行说明

本目录与「Object Recognition & Classification/2025H」为同一套 **野生动物巡查感知** 示例（香橙派 + OpenCV）。详细说明见本目录 **README.md**。

---

## 一、环境与依赖

- Python 3.7+
- `pip install -r requirements.txt`

---

## 二、纯软件模拟

```powershell
cd 本目录
pip install -r requirements.txt

python main.py --simulate
python main.py --video 你的视频.mp4
```

---

## 三、实机摄像头

```powershell
python main.py --real --cam 0 --width 640 --height 480
```

无屏：`--no-show`。参数见 **config.py**。

---

## 四、常见问题

| 现象 | 处理 |
|------|------|
| 无法打开摄像头 | 更换 `--cam` 索引；检查 USB/CSI 驱动 |
| 背景噪声大 | 调 `MOG2_VAR_THRESHOLD`、`BG_LEARNING_RATE`、形态学核大小 |
| CPU 占用高 | 降分辨率、`PROCESS_EVERY_N_FRAMES` 跳帧 |
