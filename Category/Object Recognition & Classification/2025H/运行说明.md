# 2025H 野生动物巡查 — 运行说明

本目录为 **香橙派 + OpenCV** 的野生动物运动目标感知示例。详细算法说明见 **README.md**。

---

## 一、环境与依赖

- Python 3.7+
- 本目录下执行：`pip install -r requirements.txt`
- 实机摄像头需系统能识别 `/dev/video0`（Linux）或对应设备号（Windows）

---

## 二、纯软件模拟（无需摄像头）

```powershell
cd 本目录
pip install -r requirements.txt

python main.py --simulate
python main.py --video 你的视频.mp4
```

---

## 三、实机摄像头

```powershell
python main.py --real --cam 0 --width 640 --height 480
```

- 无屏/SSH：`--no-show`
- 调参：编辑 **config.py**（`CAM_INDEX`、`PROC_WIDTH/HEIGHT`、背景建模与形态学参数等）

---

## 四、常见问题

| 现象 | 处理 |
|------|------|
| 打不开摄像头 | 换 `--cam 1/2`；Linux 检查 `v4l2-ctl --list-devices` |
| 画面全黑/过暗 | 换光源或调曝光（驱动相关 CAP_PROP） |
| 误检多 | 增大 `MIN_FOREGROUND_AREA`、加强形态学开运算 |
| 小目标漏检 | 提高分辨率、调 `MIN_BBOX_SIDE`、必要时开多尺度 |

---

## 五、提示

算法输出为每帧目标框列表，可在 **main.py** 的 `on_detection` 中接录像、串口、网络上报等。
