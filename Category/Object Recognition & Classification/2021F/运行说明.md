# 2021F 智能送药车 — 运行说明

香橙派 + OpenCV：循线、房号/标志识别、路径与返航。**详细设计见 README.md、DESIGN.md**。

---

## 一、环境与依赖

```powershell
cd 本目录
pip install -r requirements.txt
```

可选 OCR：`sudo apt install tesseract-ocr`，`pip install pytesseract`（见 README）。

---

## 二、纯软件仿真（PC 调算法）

```powershell
python run.py --sim
python run.py --sim --no-sim-window
```

---

## 三、实机运行（接硬件）

1. 在 **config/default.yaml** 中填写摄像头设备号、巡线 PID、HSV、GPIO（按你板子 **BCM** 编号）。
2. 在 **src/hal/motor.py**、**src/hal/sensor.py** 中实现真实电机与巡线传感器读取（当前默认可为 mock）。
3. 关闭 HAL mock，使用：

```powershell
python run.py --real-hal --room 2
```

- 不接摄像头仅循线：`--no-camera`
- 主循环频率：`--hz 50`

---

## 四、硬件清单（参考）

- 香橙派 + USB 摄像头
- 双轮差速底盘 + 电机驱动
- 多路巡线传感器、可选编码器
- 独立电池、共地、急停（建议）

---

## 五、常见问题

| 现象 | 处理 |
|------|------|
| `--real-hal` 后电机不动 | 需在 `src/hal/motor.py` 中实现 GPIO/PWM，并核对引脚与供电 |
| 摄像头无画面 | 改 `config` 中 `camera.device`；换 USB 口 |
| 仿真正常实机飘 | 调 PID、降低 `base_speed`、检查传感器阈值 |
