# 2021年电赛 H 题 - 用电器辨识装置（纯软件模拟）

基于 **香橙派 + OpenCV** 的纯软件模拟实现，不依赖真实电流采样硬件，用于算法验证与演示。

## 题目要点

- **任务**：根据电源线电流的电参量信息，分析并识别在用电器类别。
- **两种模式**：学习模式（存储各电器特征参量）、分析识别模式（实时指示在用电器的类别）。
- **电器**：≥7 种，其中 1～5 号电流 ≤50mA，7 号电流 >8A；1 号与 2 号电流相近但相位不同、含谐波。
- **性能**：随机增减在用电器时，实时显示可识别电器是否在用及电流特征参量，响应时间 ≤2s。

## 项目结构

```
2021H/
├── config.py              # 采样率、电器列表与模拟参数
├── appliance_simulator.py  # 电流波形模拟（基波+谐波+相位）
├── feature_extractor.py   # 特征提取（RMS、相位、功率因数、谐波比）
├── classifier.py          # 学习/识别（模板存储与最近邻匹配）
├── main.py                # OpenCV 图形界面与主流程
├── requirements.txt
└── README.md
```

## 环境与运行

### 依赖

```bash
pip install -r requirements.txt
```

需要：`numpy`、`opencv-python`、`scipy`。

### 运行

在 2021H 目录下执行：

```bash
python main.py
```

- **首次运行**：自动进入学习模式，对 1～7 号电器各采 5 段样本并保存模板到 `appliance_model.json`。
- **识别模式**：窗口实时显示当前模拟的“在用电器”、识别结果、特征参量（I_rms、相位、功率因数、H3/H5/H7）及电流波形。

### 实采样识别（CSV）

当你有 ADC/示波器导出的电流数据（CSV）时，可直接走“实采样”识别：

```bash
# 单列: 电流(mA)，或两列: 电流(mA),电压(V)
python main.py --csv samples/current_log.csv

# 指定窗口与步长（秒）
python main.py --csv samples/current_log.csv --window-s 0.2 --step-s 0.1 --no-window
```

### 操作说明

| 按键 | 功能 |
|------|------|
| **1～7** | 切换对应编号电器“开/关”（模拟随机增减电器） |
| **L** | 重新学习：清空模板并重新采集 1～7 号电器 |
| **Q** 或 **Esc** | 退出 |

## 实现要点

1. **电流模拟**：每种电器用不同 RMS、功率因数、相位、3/5/7 次谐波比例生成电流波形，满足题目对 1 号/2 号（同电流、不同相位与谐波）及 7 号大电流的设定。
2. **特征参量**：电流有效值、相对电压的相位、功率因数、3/5/7 次谐波与基波幅值比。
3. **识别**：学习阶段对各电器多次采样取平均特征作为模板；识别阶段对当前电流做特征提取，用加权欧氏距离做最近邻分类。
4. **响应时间**：软件模拟下每帧约 0.2s 数据，识别在 2s 内完成，满足题目要求。

## 扩展建议

- 接入真实 ADC 采样（如通过香橙派 SPI/I2C 读取电流芯片）替换 `appliance_simulator` 的合成波形。
- 多电器同时用电时，可增加分解算法（如 NILM）或更多特征以提高“随机增减电器”下的识别稳定性。
- 界面需显示中文时，可用 OpenCV + PIL 绘制文字或改用 PyQt/Tk 等带中文支持的 GUI。

## 实机硬件清单（建议）

- 香橙派/树莓派
- 电流采样模块（CT/霍尔 + ADC，如 ADS1115/INA219 等）
- 可选电压采样通道（用于更稳相位/PF 估计）
- 隔离电路与安全保护（强烈建议）
