# 2021H 用电器辨识 — 运行说明

本题为 **电流特征 + 分类器** 识别用电器。默认界面为 **纯软件模拟电流波形**；也可对 **CSV 实采样数据** 做离线识别。详见 **README.md**。

---

## 一、环境与依赖

```powershell
cd 本目录
pip install -r requirements.txt
```

依赖：`numpy`、`opencv-python`、`scipy`。

---

## 二、模拟模式（默认）

```powershell
python main.py
```

- 首次运行若无模型，会自动学习并生成 **appliance_model.json**。
- 按键 **1～7** 切换模拟「在用电器」；**L** 重新学习；**Q/Esc** 退出。

---

## 三、CSV 实采样识别

CSV 格式：

- **单列**：电流（mA），采样率需与 **config.py** 中 `SAMPLE_RATE` 一致。
- **两列**：`电流mA,电压V`（逗号分隔）。

```powershell
python main.py --csv samples/current_log.csv
python main.py --csv samples/current_log.csv --window-s 0.2 --step-s 0.1 --no-window
```

---

## 四、实机硬件（扩展）

- 电流互感/霍尔 + ADC（如 INA219、ADS1115 等）
- 可选电压通道用于相位/功率因数
- **强电侧务必隔离与防护**，按规范接线

实机数据可先写入 CSV，再用 `--csv` 验证算法，再在代码中替换 `appliance_simulator` 为实时采样。

---

## 五、常见问题

| 现象 | 处理 |
|------|------|
| CSV 报错或形状不对 | 检查分隔符为英文逗号；列数与说明一致 |
| 识别率低 | 重新学习（L）；检查采样率是否与 `SAMPLE_RATE` 一致 |
| 需要中文界面 | README 中建议 OpenCV+PIL 或换 GUI 框架 |
